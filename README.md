# TABLEAU ANALYTICS PORTFOLIO: SALES & SUPPLY CHAIN OPTIMIZATION

This repository showcases a Tableau analytics project built using Tableau Desktop and Tableau Prep Builder, focusing on Global Sales Performance and Supply Chain Efficiency. It provides interactive dashboards, automated KPIs, and analysis for better decision-making.

==============================
# KEY FEATURES

Interactive Tableau Dashboards

Advanced KPI Tracking

Integrated Data Prep (.tfl)

Geospatial and Time-Series Analysis

Packaged Workbook for easy use

==============================
# DATA FOCUS

Sales Dataset Includes:

Revenue and Profit

Product Categories

Customer Segments

Order and Ship Dates

Geographic Regions

Supply Chain Dataset Includes:

Inventory Levels

Stock-Out Rates

Shipping Costs

Delivery Lead Times

Supplier Performance

==============================
# ANALYSIS PERFORMED

Data Preparation and Quality

Sales Performance Analytics

Supply Chain Optimization

Profitability Mapping

Custom Visualizations

==============================
# KEY INSIGHTS (IMPORTANT)

Profit Margin Optimization

Inventory Reduction Opportunities

Regional Sales Drivers

Fulfillment Efficiency

==============================
# FINAL CONCLUSION

This project combines Sales and Supply Chain data into a unified Tableau framework.
It converts raw datasets into clear, actionable insights that support strategic and operational decisions.

==============================
# INCLUDED FILES

Tableau_supply_chain_analysis.twbx

Tableau_sales_Analysis.twb

Tableau_Dashboard.tfl

==============================
# HOW TO USE

Clone the repository

Open .twbx files in Tableau Desktop or Tableau Reader

Open .twb file for editing (requires data source connection)

Open .tfl in Tableau Prep Builder
